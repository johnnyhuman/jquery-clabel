jQuery cLabel

compact <label> plugin

jQuery cLabel plugin converts all <label>s of selected text <input>s into the <input value="label_text">. It's less than 1kb in size (670 bytes minified) and was created to bring more comfort to users.

Installation

This plugin requires jQuery JavaScript library.

Download jQuery and include it in the head of your HTML document.
Alternatively you can include it using jQuery CDN by adding the following code: 

	<script src="http://code.jquery.com/jquery-1.5.2.min.js"></script>

Download cLabel plugin and include it in the head of your HTML document after jQuery library.

	<script src="js/jquery.clabel.js"></script>

Add the following script after the previous two:

    <script type="text/javascript">
        $(document).ready(function() {
            $(".clabel").clabel();
        });
    </script>

Add the "clabel" class to <input> text elements to convert their <label>s into the <input value="label_text">. Make sure that the labels have correct "for" attributes. 

	<label for="q">Search:</label>
	<input type="text" name="q" id="q" class="clabel" />

Write CSS code to make your forms look nice with JS enabled as well as without it.

Enjoy the result and happines of your website users.
